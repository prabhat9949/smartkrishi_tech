# SmartKrishi - Minimal Skeleton Project

This is a minimal skeleton Android Studio project created from the PDF you uploaded.
It contains core files and a basic Compose setup so you can open it in Android Studio.

**Next steps**
- Open Android Studio → Import project → select this folder.
- Update `Constants.BASE_URL` with your backend endpoint.
- Add more features/files from your PDF into the package `com.smartkrishi`.

Files included:
- settings.gradle.kts, build.gradle.kts
- app module with build.gradle.kts
- Application class, MainActivity with Compose
- Simple DI NetworkModule, ApiService ping endpoint
- Basic theme and resources
